CISAがAIを悪用したフィッシングキャンペーンに関する勧告を発表

米国サイバーセキュリティインフラセキュリティ庁（CISA）は、生成AIを活用した高度なフィッシングキャンペーンについて新たな勧告を発表しました。攻撃者はLLMを使用してターゲットごとに最適化された説得力の高いメールを自動生成しており、従来の検知手法では捕捉が困難になっています。特に金融・医療・重要インフラ分野の組織が標的とされており、多要素認証の導入と従業員教育の強化が推奨されています。

示唆・学びAIによるフィッシングの自動化・個別最適化は防御側のコスト非対称性を大きく悪化させる。従来のシグネチャベース検知から行動・文脈分析への転換が急務。

研究者がエンタープライズLLMエージェントへのプロンプトインジェクション攻撃を実証

大学の研究チームが、エンタープライズ向けLLMエージェントに対する新たなプロンプトインジェクション手法を実証しました。この攻撃は悪意のあるコンテンツをドキュメントやWebページに埋め込むことでAIエージェントを操作し、機密データの外部流出を引き起こします。実験では主要な商用AIシステム3製品で再現に成功しており、業界全体に影響する深刻な問題として注目されています。

示唆・学びLLMエージェントの自律性が高まるほどプロンプトインジェクションのリスクも拡大する。入力サニタイズと最小権限原則の徹底がエージェント設計の必須要件になっている。

示唆・学びAI RMF 2.0の敵対的攻撃対策ガイダンスは事実上の業界標準になりうる。調達・契約要件への組み込みを先行して検討する価値がある。

ランサムウェアグループがAIによる脆弱性自動発見を開始

複数の脅威インテリジェンスレポートにより、ランサムウェアグループがAIを活用した脆弱性自動発見ツールの使用を開始したことが明らかになりました。これらのツールはコードベースを自動解析してゼロデイ脆弱性の候補を特定する能力を持ち、従来の手動調査と比べて発見速度が数十倍に向上していると報告されています。パッチ適用の遅延が以前にも増して重大なリスクになっています。
